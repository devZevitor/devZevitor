## 💻 Projeto em Destaque

**EletroConsertos - Sistema de Gerenciamento de Pedidos**

🔗 **Repositórios:** [Backend](LINK-BACKEND-REPO) | [Frontend](LINK-FRONTEND-REPO)  
🌐 **Deploy:** [Aplicação Web](https://eletroconsertos.vercel.app/) | [API Docs](https://surprising-zulema-interdiciplinar-fatec-fc7cbde3.koyeb.app/docs)

**Stack do projeto:**
- **Backend:** Fastify + TypeScript + Drizzle ORM + PostgreSQL (Supabase) + JWT
- **Frontend:** React + TypeScript + Vite + TanStack Query + Tailwind CSS + shadcn/ui
- **Deploy:** Vercel (frontend) + Koyeb (backend) + Docker

> Projeto interdisciplinar completo em produção, com arquitetura moderna e boas práticas de desenvolvimento full stack. Sistema para gerenciamento de pedidos de consertos com autenticação JWT, documentação Swagger e deploy automatizado.
